# Zippy Logitech - Logistics Business Intelligence Platform

> AI-powered logistics platform for the Indian market

## Project Overview

**Type:** Full-stack logistics platform with AI/ML
**Focus:** Freight matching, dynamic pricing, route optimization
**Market:** India (Tamil Nadu → pan-India expansion)

## Technology Stack

| Layer | Technology |
|-------|------------|
| **Backend** | FastAPI, PostgreSQL, Redis, Kafka |
| **AI/ML** | LightGBM, OR-Tools, DRL4Route |
| **Frontend** | React 18 + TypeScript + Tailwind |
| **Mobile** | React Native (Driver + Customer apps) |
| **Payments** | Razorpay + GST e-way bill |
| **Database** | 4,038 records (insights, metrics, vehicles) |
