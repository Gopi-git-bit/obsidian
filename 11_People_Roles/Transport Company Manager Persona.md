# Transport Company Manager Persona

## Profile

| Attribute | Description |
|-----------|-------------|
| Business Type | Fleet operator (3-50 trucks) |
| Role | Manager or Owner-operator |
| Fleet Size | Owner + partner vehicles |
| Decision Maker | Yes (for own fleet) |
| Tech Proficiency | Low-Medium |
| Key Need | Consistent loads, fast payment |

## Goals

- Maximize fleet utilization
- Minimize deadhead/empty runs
- Ensure timely payments
- Reduce driver-related issues
- Grow business sustainably

## Pain Points

### Primary
- **Empty runs** - No return load visibility
- **Payment delays** - Cash flow stress
- **Customer acquisition** - Finding reliable shippers
- **Driver management** - Retention, performance

### Secondary
- Vehicle maintenance costs
- Fuel price fluctuations
- Compliance paperwork
- Competition from aggregators
- Weak return-load visibility in Tier-2 and Tier-3 corridors

## Behavior Patterns

| Pattern | Insight |
|---------|---------|
| Price-sensitive | Compare rates across platforms |
| Relationship-oriented | Trust built over time |
| Cash-dependent | Prefer immediate payment |
| Risk-averse | Avoid uncertain contracts |
| Region-sensitive | Values lanes with predictable return economics |

## Objections

| Objection | Response |
|-----------|----------|
| "Commission too high" | Show volume benefits |
| "Platform takes my customers" | Offer exclusivity options |
| "Payment takes too long" | Provide instant payment option |
| "My drivers can't use apps" | Offer simple interface + support |

## Quote to Remember

*"I don't care about apps. I care about loads that pay on time and don't create problems."*
