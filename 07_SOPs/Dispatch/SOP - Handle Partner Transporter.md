# SOP - Handle Partner Transporter

## Trigger

Partner transporter interaction - onboarding, order assignment, or issue resolution.

## Preconditions

- Partner registered and verified
- Partnership terms agreed
- Order available for assignment

## Steps

### Step 1: Partner Verification

```
1.1 Verify partner credentials
1.2 Check vehicle certifications
1.3 Review partnership terms
1.4 Confirm insurance coverage
```

### Step 2: Order Assignment

```
2.1 Share order details with partner
2.2 Confirm vehicle and driver availability
2.3 Agree on pricing and terms
2.4 Send official assignment
```

### Step 3: Coordination

```
3.1 Monitor acknowledgment
3.2 Track pickup confirmation
3.3 Follow transit progress
3.4 Maintain communication channel
```

### Step 4: Issue Handling

```
4.1 Receive issue notification
4.2 Assess impact on delivery
4.3 Coordinate resolution
4.4 Escalate if unresolved
```

### Step 5: Completion

```
5.1 Verify delivery completion
5.2 Confirm POD received
5.3 Process payment to partner
5.4 Update partner performance record
```
