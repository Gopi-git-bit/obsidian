# SOP - Escalate Delayed Shipment

## Trigger

Shipment delay exceeds acceptable threshold and requires management intervention.

## Preconditions

- Delay identified and assessed
- Initial handling attempted
- Escalation criteria met

## Steps

### Step 1: Confirm Escalation

```
1.1 Verify delay threshold crossed
1.2 Document initial handling attempts
1.3 Prepare escalation summary
1.4 Assign severity level
```

### Step 2: Notify Management

```
2.1 Contact operations manager
2.2 Share delay details and impact
2.3 Provide customer sensitivity
2.4 Note time sensitivity
```

### Step 2: Executive Notification

```
2.1 IF critical (>12 hours): VP notification
2.2 IF VIP customer: Dedicated handler
2.3 IF legal risk: Legal team alert
```

### Step 3: Prepare Options

```
3.1 Identify resolution paths
3.2 Calculate cost of each option
3.3 Recommend preferred action
4.4 Prepare customer communication draft
```

### Step 4: Execute Decision

```
4.1 Get approval for action
4.2 Implement resolution
4.3 Update customer with resolution
4.4 Monitor resolution progress
```

### Step 5: Post-Resolution

```
5.1 Confirm delivery completed
5.2 Apply compensation if promised
5.3 Document escalation outcome
5.4 Schedule review meeting
```

## Escalation Matrix

| Delay Duration | Customer Type | Escalation To |
|----------------|---------------|---------------|
| >4 hours | Standard | Team Lead |
| >8 hours | Standard | Manager |
| >4 hours | Premium | Manager |
| >2 hours | VIP | VP |
| Any | Legal threat | Legal |
