## Daily Workflow

1. Capture raw notes in `01_Inbox`.
2. Convert useful content into concepts, algorithms, scenarios, or SOPs.
3. Link new notes to at least two related notes and one relevant hub.
4. Use hubs to navigate, not folders alone.

## Contribution Rules

1. Start from [[Note Templates]].
2. Add frontmatter that reflects the note type and status.
3. Link upward to a relevant hub.
4. Add at least two meaningful related links.
5. If a referenced note does not exist yet, either create it soon or replace the link with an existing note.
