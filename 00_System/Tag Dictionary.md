## Tagging Rules

### DO
- Use tags consistently
- Keep tag list short (max 20 active tags)
- Add tags in frontmatter, not body text
- Use lowercase for tags

### DON'T
- Don't tag every word
- Don't create tags for single notes
- Don't use tags instead of links
- Don't mix naming conventions

## Tag Workflow

1. When creating a note, add 2-3 relevant tags
2. Tags go in frontmatter, not body
3. Review monthly for cleanup
4. Remove unused tags

---

*Tags help AI retrieval. Links help human navigation. Use both.*
