# Commission vs Flat Fee

## Commission Model

### Definition
Percentage-based fee deducted from each transaction (freight amount).

### Example
- Shipper pays ₹25,000 for shipment
- Platform takes 15% = ₹3,750
- Transporter receives ₹21,250

### Pros

| Benefit | Description |
|---------|-------------|
| Aligned incentives | Platform profits when shipper profits |
| Scalable | Grows with transaction value |
| Simple | Easy to understand |

### Cons

| Drawback | Description |
|----------|-------------|
| Price transparency | Shipper sees deduction |
| Rate pressure | Shippers push for lower commission |
| Volume dependency | Revenue fluctuates |

### Typical Rates (India)
- Aggregators: 10-20%
- 3PL: 8-15%
- Freight marketplaces: 5-12%

## Flat Fee Model

### Definition
Fixed amount charged per shipment regardless of value.

### Example
- Shipper pays ₹25,000 for shipment
- Platform takes ₹500 flat fee
- Transporter receives ₹24,500

### Pros

| Benefit | Description |
|---------|-------------|
| Predictable | Easier for planning |
| Transparent | No hidden math |
| Works for high-value | Proportionally cheaper |

### Cons

| Drawback | Description |
|----------|-------------|
| Small shipments | Unprofitable for platform |
| No upside | Doesn't scale with value |
| Competitive pressure | Hard to justify |

### Typical Rates (India)
- Urban delivery: ₹20-50
- Regional: ₹100-300
- Long-haul: ₹500-1000

## Hybrid Model

### Definition
Combination of lower commission + flat fee

### Example
- 8% commission + ₹200 flat
- ₹25,000 shipment: ₹2,000 + ₹200 = ₹2,200

### Benefits
- Base revenue guaranteed
- Upside participation
- More transparent

### Modern Platform Extension

Many logistics platforms now separate:

- transport price
- customer-segment discount
- platform fee
- VAS charges
- insurance
- tax

This improves quote transparency and makes hybrid monetization easier to explain operationally.

## Decision Framework

| Consider | Commission Better | Flat Fee Better |
|---------|------------------|-----------------|
| Shipment value | Variable | Predictable |
| Transaction frequency | High volume | Low volume |
| Market position | Competing on service | Competing on price |
| Margins | High margin business | Thin margin |

## Practical Zippy Implication

- Driver-side monetization can remain commission-oriented.
- Transport-company monetization can remain flat-fee-oriented.
- Customer-facing pricing can still include transparent service, insurance, and platform layers without changing settlement logic.
