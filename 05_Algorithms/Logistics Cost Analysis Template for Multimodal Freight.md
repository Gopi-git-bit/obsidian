# Logistics Cost Analysis Template for Multimodal Freight

## Purpose

Use this note as the standard quotation and route-decision cost sheet for multimodal freight.

The core logistics-cost principle from LSCM Unit-2 is:

> Do not minimize one cost item alone. Minimize the total logistics system cost.

Do not ask only:

```text
Which transporter gives the cheapest quote?
```

Ask:

```text
Which option gives the lowest total cost after transport, handling, inventory, delay, damage, documentation, and customer-service cost?
```

That is the proper logistics cost brain. Tiny calculator, big boots.

---

## 1. Should I Use A Common Template?

Yes.

Use one common template for every shipment so that all routes can be compared fairly.

But customize the template depending on:

- transport mode
- cargo type
- urgency
- distance
- customer promise
- risk level
- export/import requirement
- temperature requirement
- handling complexity

## Core Rule

```text
Use a common base template + cargo/mode-specific cost add-ons.
```

Example:

| Shipment Type | Base Template | Add-On |
| --- | --- | --- |
| Road-only freight | Yes | fuel, toll, driver, waiting |
| Road + rail | Yes | terminal, loading, rail booking, last-mile |
| Export shipment | Yes | customs, CFS, port, documentation |
| Pharma | Yes | cold chain, sensors, compliance |
| Electronics | Yes | security, insurance, packaging |
| FMCG | Yes | warehouse, picking, returns |
| Bulk cargo | Yes | loading equipment, storage, demurrage |

---

## 2. Why Common Template Is Needed

A common template helps compare:

```text
Road-only vs Road + Rail vs Road + Port vs Road + Air
```

Without a common template, the startup may choose the cheapest visible freight rate but miss hidden costs.

## Example

| Cost Item | Road Only | Road + Rail |
| --- | ---: | ---: |
| Freight quote | Rs 80,000 | Rs 55,000 |
| Handling | Rs 2,000 | Rs 8,000 |
| Terminal waiting | Rs 0 | Rs 7,000 |
| Delay cost | Rs 5,000 | Rs 10,000 |
| Documentation | Rs 1,000 | Rs 3,000 |
| Total cost | Rs 88,000 | Rs 83,000 |

Road + rail is still cheaper, but not by Rs 25,000. Real saving is Rs 5,000.

That is why total cost analysis matters.

## 3. Core Logistics Cost Categories

LSCM Unit-2 material identifies logistics cost elements such as transportation cost, ordering cost, loss and damage, perishability, inventory cost, warehousing, handling, documentation, and customer-specific servicing costs. It also emphasizes that management should minimize total logistics cost, not each component separately.

## Common Cost Categories

| Cost Category | Meaning | Example |
| --- | --- | --- |
| Transport cost | Cost paid to carrier/mode | truck, rail, air, sea |
| Ordering/admin cost | Cost to arrange shipment | staff time, booking, billing |
| Handling cost | Loading/unloading/transshipment | labour, forklift, terminal |
| Warehousing cost | Storage and space cost | rent, pallet, cold room |
| Inventory carrying cost | Cost of goods sitting in transit/storage | capital cost, shelf-life loss |
| Loss/damage cost | Expected damage/loss risk | claims, replacement |
| Perishability cost | Value loss from delay/deterioration | seafood, pharma, vegetables |
| Packaging cost | Protective or standard packaging | crates, pallets, insulation |
| Documentation cost | Invoice, e-way bill, customs, POD | export docs |
| Customer service cost | Special service required by customer | special packaging, returns, credit |
| Delay cost | Cost of missing time window | penalty, reattempt, missed port cut-off |
| Return/refusal cost | Reverse logistics or failed delivery | customer rejects goods |

## 6. Total Logistics Cost Formula

```text
Total Logistics Cost =
Transport Cost
+ Handling Cost
+ Warehousing Cost
+ Inventory Carrying Cost
+ Documentation Cost
+ Packaging Cost
+ Insurance Cost
+ Delay Cost
+ Risk / Failure Cost
+ Admin Cost
```

## Cost Per Tonne-Km

```text
Cost per tonne-km = Total Logistics Cost / (Weight in tonnes x Distance in km)
```

## Cost Per Shipment

```text
Cost per shipment = Total Logistics Cost / Number of shipments
```

## Cost Per Unit

```text
Cost per unit = Total Logistics Cost / Number of units
```

## 7. Activity-Based Costing Template

Activity-based costing is useful because it assigns cost to actual activities instead of blindly spreading overhead. The source explains ABC as identifying activities and assigning costs to products/services based on actual consumption, giving better visibility into cost drivers.

| --- | --- | ---: | ---: | ---: |

| Picking/packing | order lines | Rs/line |  |  |
| Documentation | documents | Rs/document |  |  |
| Customer support | service tickets | Rs/ticket |  |  |
| Returns | returns | Rs/return |  |  |
| Claims | claims | Rs/claim |  |  |

## ABC Formula

```text
Activity Cost = Activity Rate x Activity Quantity
```

## Example

```text
Loading Cost = Rs 250 per tonne x 10 tonnes = Rs 2,500
Documentation Cost = Rs 150 per document x 8 documents = Rs 1,200
Storage Cost = Rs 40 per pallet-day x 20 pallet-days = Rs 800
```

## 8. Customer Profitability Template

Customer profitability analysis matters because two customers may buy the same amount but cost very differently to serve because of order complexity, transport, handling, special packaging, returns, credit period, and service requirements.

## Customer Profitability Formula

```text
Customer Profit =
Revenue from Customer
- Product / Service Cost
- Transport Cost
- Warehousing Cost
- Handling Cost
- Documentation Cost
- Returns / Refusals
- Discounts / Promotions
- Credit Cost
- Special Service Cost
```

## 9. Direct Product Profitability Template

The Unit-2 material gives this DPP logic:

```text
Sales Price - Purchase Cost = Gross Margin

Warehouse Cost + Transportation Cost + Store Cost = Direct Product Cost

Gross Margin - Direct Product Cost = Direct Product Profitability
```

## 10. Mode Comparison Template

The newer DPIIT/NCAER logistics-cost assessment estimates India's logistics cost at 7.97% of GDP in FY 2023-24 and reports rail as significantly cheaper per tonne-km than road in mode-wise comparisons. Economic Times reported rail at around Rs 1.96 per tonne-km and road at around Rs 11.03 per tonne-km, with air the most expensive.

Use these as benchmark references, not automatic prices.

| Mode | Base Cost | Hidden Cost | Best Use |
| --- | --- | --- | --- |
| Road | high per tonne-km | toll, fuel, waiting, empty return | short/urgent/first-last mile |
| Rail | low line-haul cost | terminal, handling, first-last mile | long-haul bulk/container |
| Air | very high | airport handling, security | urgent high-value goods |
| Water/coastal | low for heavy cargo | port delay, slower time | non-urgent port-linked cargo |

## Mode Decision Rule

```text
Do not choose mode by line-haul rate alone.
Choose mode by total landed logistics cost.
```

## 13. Pricing Template For Startup

The customer price should not equal carrier cost.

## Startup Price Formula

```text
Customer Price =
Total Logistics Cost
+ Risk Buffer
+ Service Margin
+ Working Capital Buffer
```

## Suggested Buffers

| Service Type | Margin / Buffer |
| --- | ---: |
| Simple road-only | 8-12% |
| Repeated lane contract | 5-10% |
| Multimodal coordination | 12-18% |
| Export/document-heavy shipment | 15-25% |
| Cold chain / pharma | 18-30% |
| Guaranteed SLA | 20-35% |

## 14. What Cost Template Must Reveal

A good template must reveal:

| Question | Why It Matters |
| --- | --- |
| Which cost is largest? | Shows biggest saving opportunity |
| Which cost is hidden? | Prevents underpricing |
| Which route is cheapest? | Basic comparison |
| Which route is most reliable? | Protects delivery promise |
| Which customer is profitable? | Avoids bad accounts |
| Which activity drives cost? | ABC improvement |
| Which mode is overused? | Helps shift to rail/port |
| Which cost is non-operational waste? | Shows what to cut |

## 16. Final Decision Rule

Use a common template, but never use only one number.

For every route, compare:

```text
Total logistics cost
+ delivery time
+ reliability
+ risk
+ margin
```

The best route is not:

```text
lowest freight quote
```

The best route is:

```text
lowest reliable total cost with acceptable service quality
```

That is the difference between a transport broker and a logistics decision engine.

---

## Practical Answer

Yes, use a common template. Make it the standard quotation and decision sheet.

But let the template expand depending on the shipment type.

Road-only freight may need 10 cost fields. Pharma export through port may need 40.

Same skeleton, different armor.
