# Scenario - Hazardous Material Transport

## Situation

Customer needs to transport hazardous materials (chemicals, flammable goods, batteries) that require special permits, vehicles, and handling.

## Variables

| Variable | Value | Notes |
|----------|-------|-------|
| Hazmat class | Various | Determines requirements |
| Permits required | Yes | State-specific |
| Vehicle type | Certified | Hazmat-certified |
| Documentation | Extended | Compliance docs |

## Recommended Response

```
1. VERIFY hazmat class and requirements
2. CHECK vehicle certification for type
3. OBTAIN required permits (if platform handles)
4. COORDINATE with certified transporter
5. ENSURE emergency equipment available
6. COMMUNICATE handling instructions
```
